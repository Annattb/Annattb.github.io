几丁质与壳聚糖

贝类废物中富含几丁质 ,几丁质降解后得到产物壳聚糖

壳聚糖生物活性丰富但具有明显缺陷

壳聚糖的局限性

分子量过大、水溶性较差 生物活性丰富但在水溶液中的作用较小

壳寡糖（COS）的优越性

分子量小、黏度低、水溶性强

具有很好的抗真菌、抗病毒活性

保留了壳聚糖所具有的多种生物活性

如抗感染、抗肿瘤、免疫调节

壳聚糖酶的酶学性质

最适温度

4. 0-8. 0

最适pH值

30～60 ℃

活性金属离子影响

Mn2+、Mg2+、Ca2+、Zn2+、Cu2+、Co2+ 促进降解壳聚糖 K+、Fe2+、Hg2+、Ni2+ 抑制降解壳聚糖
